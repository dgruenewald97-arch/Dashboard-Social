Einmalig. Danach holt das Dashboard die Daten auf Knopfdruck. Stand-API: v25.0.

  1. Instagram = Business-Konto und mit deiner Facebook-Seite verbunden (Instagram → Einstellungen → Verknüpfte Konten).
  2. Öffne den Graph API Explorer, App wählen → „Generate Access Token". Rechte: pages_read_engagement, pages_show_list, instagram_basic, instagram_manage_insights, read_insights.
  3. Page-ID: Abfrage me/accounts ausführen → bei deiner Seite steht "id".
  4. Page-Token: In derselben Antwort steht pro Seite ein "access_token" — kopieren.
  5. Beides unten einfügen und „Verbinden & Testen" klicken.
Für einen Token, der nicht abläuft: Business Manager → Systembenutzer. Details in ENDPOINTS.md.
